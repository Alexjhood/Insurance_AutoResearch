# data/raw/

Raw freMTPL2 frequency and severity files go here (CSV or Parquet). The loader
auto-discovers files by filename substring: `freq` for the frequency table and
`sev` for the severity table.

## Two ways to populate this directory

**For testing / smoke runs (no network required):**

```bash
python scripts/generate_synthetic_data.py
```

Writes `freMTPL2freq_synthetic.parquet` (5 000 rows) and
`freMTPL2sev_synthetic.parquet`. Fully deterministic; safe for CI.

**For real research:**

```bash
python scripts/fetch_fremtpl2.py
```

Downloads ~678 K rows from OpenML (ids 41214 and 41215) and writes
`freMTPL2freq.parquet` and `freMTPL2sev.parquet`. Requires network access.

## Licensing

freMTPL2 is distributed under the CASdatasets / OpenML terms. For academic use,
cite Charpentier (2014). Synthetic data generated by `generate_synthetic_data.py`
has no license restriction.

## Expected columns

See `src/autoresearch/data/anonymise.py` for the authoritative column mapping.
The frequency table must include `IDpol`, `ClaimNb`, `Exposure`, and the vehicle
and driver feature columns. The severity table must include `IDpol` and
`ClaimAmount`.
